## What languages are used?

#### This project is built with the following tech stack:
**FE:** 
- Reactjs
- Typescript
- SCSS
- Tailwind

**BE:** 
- Nodejs
- Strapi CMS
- Stripe checkout


## How to Run e-commerce site locally
client root: `npm i && npm run start-dev`

server root: `npm i && npm run develop`
